\section{Waves \& Rays}
\frame{
  $q=$ causal potential for acoustic waves radiating from isotropic point source
  at $\bx_s \in \bR^3$ with waveform $f(t)$ - solves
  \[
    \square q(\bx,t;\bx_s)  = \left(\frac{1}{\kappa}\frac{\partial^2 q}{\partial t^2} - \frac{1}{\rho}\nabla^2 q\right)(\bx,t;\bx_s)=
    f(t) \delta(\bx-\bx_s),\,q=0 \mbox{ for } t \ll 0
  \]
  Pressure $p = \rho \partial_t q$ - depends on bulk modulus $\kappa(\bx)$ 

  {\color{blue} Dynamic Inverse Problem:} given pressure history
  $\{d(\bx_r,t; \bx_s)\}$ for source, receiver
  locations $(\bx_s, \bx_r) \in X_{sr}$, find $\kappa$ so that $p[\kappa]
  \simeq d$.

  Regard density $\rho \in \bR$, source waveform $f \in C_0^{\infty}(\bR)$ as known - highly
  idealized!

  Many variants, large literature since '70's, many apps 
}

\frame{
  High frequency asymptotics (geometric optics/acoustics,...): in
  HF limit, travel time of wave from source at $\bx_s$ to receiver
  at $\bx_r$ is {\em geodesic distance} $\tau[\kappa](\bx_s,\bx_r)$ in metric $ c^{-2}\sum_i dx_i
  \otimes dx_i$, $c = \sqrt{\kappa/\rho}$ = sound speed

{\color{blue} Kinematic Inverse Problem:} given times
    $t(\bx_s,\bx_r)$, find $\kappa$ so that $\tau[\kappa](\bx_s,\bx_r)
    = t(\bx_s,\bx_r)$ for $(\bx_s,\bx_r) \in X_{sr}$.

Huge literature - boundary rigidity problem (find $c$ in $\Omega
\subset \bR^n, X_{sr} \subset \partial \Omega$), many apps (seismic
tomography,...)

Complication: multiple geodesics (``rays'') may connect
source/receiver points - which $\tau$ to match?
}

\frame{
  Common computational approach;: pose as
  optimization, eg. least squares - use iterative algorithm to find
  $\kappa$ to (approximately) minimize
  \[
  J_{\rm FWI}[\kappa,d] = \sum_{(\bx_s,\bx_r) \in X_{sr}} \int\,dt\,
|p[\kappa](\bx_r,t;\bx_s) - d(\bx_r,t;\bx_s)|^2
\]
(``Full Waveform Inversion'') or
\[
  J_{\rm TT}[\kappa,t] = \sum_{(\bx_s,\bx_r) \in X_{sr}}
|\tau[\kappa](\bx_s,\bx_r) - t(\bx_s,\bx_r)|^2
\]
(``Traveltime Tomography'').
}

\frame{
Good news, bad news...

For FWI,
\begin{itemize}
\item seismology, ultrasonic imaging, ocean acoustics,... the
  data are pressure (or ...) histories, so FWI is the natural
  approach, BUT...
\item unless initial $\kappa$ predicts traveltimes accurately
  (``within a half-wavelength''), Newton-like iterations tend to
  stagnate at incorrect final $\kappa$ with big data prediction
  error - behaviour of optimization is wavelength-related (``cycle-skipping'')
\end{itemize}

For TT,
\begin{itemize}
\item $J_{TT}$ {\em a priori} unrelated to wavelength, optimizations tend to
  converge to good fit, BUT...
\item data t are not directly measured, must be inferred
  (``picked'') from waveform data - mostly automated in
  industrial codes, but can be noise-sensitive
\end{itemize}

}

\frame{
Practical remedy for cycle skipping: pick times $t(\bx_r,\bx_s)$,
minimize $J_{\rm TT}$, initiate minimization of $J_{\rm FWI}$
from result, low $\rightarrow$ high frequency continuation - widespread use, but many attempts at alternatives

Warner et al. (2012, ...): FWI fails because initial prediction
data $p[k]$ places waves at wrong times - so {\em move them} to make
residual small

Use a {\em filter} (convolution kernel) $u_{\sigma}(\bx_r,t;\bx_s)$, solution of
\[
  \min_u \|p[\kappa] * u_{\sigma} - d\|^2 + \sigma \|u_{\sigma}\|^2
\]
regularization weight $\sigma > 0$ TBD. NB: $u_{\sigma}$ depends on
$\kappa, d$

($*$ = convolution in $t$, $\|\cdot\| = L^2$ norm)
}

\frame{
  illustration: resim0, uest0
}

\frame{
  BUT desiired outcome is $p[\kappa] \approx d$ which is $p[\kappa]*u
  \approx d$ with $u = \delta(t)$

  Since $t\delta(t) =0$, try minimizing
  \[
    J_{{\rm AWI},\sigma}[\kappa,d] = \frac{\|tu_{\sigma}[\kappa,d]\|^2}{\|u_{\sigma}[\kappa,d]\|^2}
  \]
  
  Many successful results reported - overcomes cycle skipping! (at
  least sometimes...) 

  Why should this work? Answer must lie in HF asymptotics...

  {\color{blue} Under some conditions $ J_{{\rm AWI},\sigma}[\kappa,d]
  \rightarrow J_{TT}[\kappa,t]$ as data wavelength $\rightarrow 0$}
}

\frame{
  Assume:

  {\color{blue} only one geodesic connecting points near sources and
    receivers (simple metric);}

  once a geodesic leaves vicinity of $X_{sr}$, it never returns;

  exponential time decay of solutions;

  asymptotics via family of source waveforms $\{\fl(t) =
  \sqrt{\lambda}f_1(t/\lambda)\}$, $f_1 \in \partial_t^2 C_0^{\infty}(\bR)$,
  corresponding pressures $\pl[\kappa]$.
  
  noise-free data $\dl^* = \pl[\kappa^*]$.
}

\frame{
  Then:

  \[
    \|\pl[\kappa]*\us[\kappa,\dl^*]-\dl^*\| \ll \|\dl^*\| \Leftrightarrow  \sigma \approx
    \mbox{ const. } \lambda
  \]
  
  so assume $\sigma\propto \lambda$ as $\lambda \rightarrow 0$, then:

  $\us[\kappa,\dl^*], t\us[\kappa,\dl^*] \in L^2(\bR \times X_{sr})$, so
  $J_{\rm AWI,\sigma}$ is well-defined, and for sufficiently small
  $\sigma$ (and $\lambda$),
  
  \[
    |J_{\rm AWI,\sigma}[\kappa,\dl^*] - J_{\rm TT}(\kappa,t^*)| \le C\lambda^2
  \]
  
  ($t^*(\bx_s,\bx_r) = \tau[\kappa^*](\bx_s,\bx_r)$)
}

\frame{
  Why?

  Hadamard expansion of Green's function (3D) $\Rightarrow$ for solution
  of $\square q = f(t) \delta(\bx-\bx_s)$ in simple metric domain
  (unique geodesic connects $\bx_s$ to $\bx$):
  \[
    q(\bx,t;\bx_s) = a(\bx.\bx_s)f(t-\tau(\bx,\bx_s)) +
    \int_{\tau(\bx,\bx_s)}^t ds\,b(\bx,t-s;\bs_s)f(s)
  \] 
  $a$, $b$ smooth for $\bx \ne \bx_s$ - ``No Return'' $\Rightarrow$
  global in $t$ 

  So $p[\kappa] = a \partial_t f(t-\tau) + ...$

  (Courant \& Hilbert '62, Friedlander '75, H\"{o}rmander '85, Wei et al. '24)
}

\frame{
  ignoring ``...'', noise free data $d^* = p[\kappa^*] \approx
  a^*f(t-\tau^*)$  ($a^*=a[\kappa^*]$ etc.)

  so
  \[
    \us = \mbox{argmin}_u (\|a \partial_tf * u(t-\tau) - a^*
    \partial_tf (t-\tau^*)\|^2)
  \]
  Solve via Fourier transform, obtain
  \[
    = \frac{a^*}{a}g_{\sigma}(t-(\tau-\tau^*)),\, \hat{g}_{\sigma}(\omega) =
    \frac{a^2 \omega^2|\hat{f}(\omega)|^2}{a^2
      \omega^2|\hat{f}(\omega)|^2 + \sigma}
  \]

  Time error $\tau-\tau^*$ appears explicitly! (also $g_{\sigma}$ is even...)
 
}

\frame{
  so
  \[
    \|t\us\|^2 = \frac{(a^*)^2}{a^2}( (\tau-\tau^*)^2\|g_{\sigma}\|^2
    + \|tg_{\sigma}\|^2 )
  \]
  \[
    \|\us\|^2 = \frac{(a^*)^2}{a^2}\|g_{\sigma}\|^2
  \]
  whence
  \[
    J_{\rm AWI,\sigma} \approx
    (\tau-\tau^*)^2 + \frac{\|tg_{\sigma}\|^2}{\|g_{\sigma}\|^2} 
  \]
  choose $f = \fl$ and $\sigma \propto \lambda$ then
  \[
    = (\tau-\tau^*)^2 + O(\lambda^2)
  \]

  $O(\lambda^2)$ estimates for ``...'', other fine print - S. arXiv '24 
}

\frame{
  Assumptionw for Bulk moduli $\log \kappa \in M \subset
  C^{\infty}(\bR^3) \cap L^{\infty}(\bR^3)$ and $X_{sr}$:

  1.{\color{blue} Large time decay:} for  $\log \kappa \in M, n \ge 0$, there are $C,
  t_0,\delta > 0$ so that for any $f \in C^{\infty}_0(\bR), 0 \le k
  \le n$, solution $q$ of $\square q = f(t)\delta(\bx-\bx_s)$ satisfies
  \[
    |\partial_t^kq(\bx_r,t;\bx_s)| \le C e^{-\delta t}\|\partial_t f\|, t > t_0
  \]
  [sufficient that $\kappa - \mbox{ const.}$ has compact support, and
  all geodesics $\rightarrow \infty$ (metric is non-trapping) - Vainberg '75] 

  2. {\color{blue} Simple metric:} $X_{sr} \subset \Omega \times
  \Omega$, $\Omega \subset \bR^3$ open, bounded, unique geodesic
  connects every pair of points 

  3. {\color{blue} No return:} once a geodesic leaves $\Omega$, it
  does not return 
}

\frame{
  4.{\color{blue} Zero frequency:} family $\{\fl: \lambda > 0\}$ of source waveforms,
  \[
    \fl(t) = \sqrt{\lambda}f_1(t/\lambda), \, f_1 = \partial_t^2
    \tilde{f}_1, \tilde{f} \in C_0^{\infty}(\bR)
  \]
  ($|\hat{\fl}(\omega)| = O(|\omega|^2)$)

  Corresponding families of potentials $q_{\lambda}$, pressures
  $p_{\lambda}$

  $\lambda$ = proxy for wavelength ($\propto$ RMS wavelength)
}
