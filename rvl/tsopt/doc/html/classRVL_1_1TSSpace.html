Detailed Description

template<typename T>
class RVL::TSSpace< T >

RVL::Space subtype whose RVL::DataContainer class is a RVL::StdProductDataContainer with RVL::TSDC components.

Only attribute distinguishing this from RVL::StdProductDataContainer is that the buildDataContainer method is guaranteed to return a RVL::TSDC.
