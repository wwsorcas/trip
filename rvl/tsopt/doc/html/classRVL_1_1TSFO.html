Detailed Description

Basic class for RVL::FunctionObject s that act on a RVL::TSDC - interface for all time step functions.

Unary action because all time step functions in their basic form operate in overwrite mode, i.e. x=f(x). Subclass must define action on a RVL::TSDC - note that parent class has no nontrivial methods, so this is significant!
